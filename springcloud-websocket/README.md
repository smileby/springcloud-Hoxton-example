# springcloud-websocket

> 场景：模拟张三、李四聊天

> 项目访问地址： http://127.0.0.1:8005/

> 注意： 该demo为单机版的实现，不支持集群环境操作。在服务器集群的环境下，会涉及到WebSocket Session共享的问题
